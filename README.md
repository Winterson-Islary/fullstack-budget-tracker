### Fullstack-Budget-Tracker (POCKET-BUDDY)

- Tools:
  1.  React Router Dom.
  2.  TanStack Query.
  3.  Clerk.
  4.  TailwindCSS.
  5.  Shadcn-ui.
- Instructions:
  1. Set up the environment variables inside the **.env.local** file.
  2. Run `npm/pnpm install` in the root dir.
  3. Run `npm/pnpm run dev` to start.
